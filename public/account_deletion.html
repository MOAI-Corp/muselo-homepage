회원 탈퇴 안내

회원님의 개인정보 보호와 안전한 서비스 이용을 위해 탈퇴 전 아래 내용을 반드시 확인해 주시기 바랍니다.

1. 모바일 앱에서의 탈퇴 방법

  1. 앱 화면 우측 상단의 [설정] 아이콘을 탭합니다.
  2. [회원탈퇴] 메뉴를 선택합니다.
  3. [탈퇴하기] 버튼을 탭하면 탈퇴 처리가 완료됩니다.

중요: 탈퇴 처리는 즉시 진행되며, 탈퇴 완료 후에는 취소할 수 없습니다.

중요: 탈퇴가 완료되면 등록된 기기에 접근할 수 없습니다. 탈퇴하기 전에 아래의 안내에 따라 사용자 데이터를 삭제해 해주세요.

2. 뮤젤로 기기에서의 사용자 데이터 삭제 방법

  1. 앱 화면 우측 하단 [캔버스] 탭으로 이동합니다.
  2. 데이터 삭제를 원하는 기기를 선택하여 화면을 이동합니다.
  3. 해당 기기 화면에서 우측 상단의 [설정] 아이콘을 누릅니다.
  4. [기기 초기화] 버튼을 눌러 모든 사용자 데이터를 삭제하고 초기화를 진행합니다.

중요: 뮤젤로 제품에 전달된 데이터는 제품 내부에만 저장됩니다. 기기에 연동된 회원의 모바일 앱에서 삭제를 명령할 수 있습니다.

3. 데이터 처리 안내

탈퇴 시 삭제되는 데이터

  • 회원 계정 정보 (이메일, 비밀번호 등)
  • 프로필 정보 (닉네임, 프로필 이미지 등)
  • 개인 맞춤 설정 및 환경설정
  • 회원 업로드 데이터(사진, 그림 파일)

법적 의무에 따라 보관되는 데이터

데이터 유형 보관 기간 보관 이유
결제 내역 5년 전자상거래법에 따른 거래기록 보관
고객센터 문의 내역 3년 소비자 분쟁 해결 및 서비스 품질 개선
IP 주소, 접속 기록 3개월 정보통신망법에 따른 보관 의무

중요: 탈퇴 후에는 계정과 관련된 개인정보가 삭제되어 복구가 불가능합니다.
